Nooo Agency Website

This was my last project done in Nooo Agency: their website. The given theme was «industriousness», so the idea of machines and robots went up pretty quickly. Obviously, other concepts were explored as well.

The Landing Page Slider

It was essentially made by three main slides. Each slide was rotating onto the previous one in a clock-like movement. In order to accomplish the result, I wrote some custom CSS3 animations and transitions.

To let the whole team figure out the final result, I built up also a small tech demo (with an introductory slide). Later on I dediced to rely a little more on a slider that could provide fallbacks for older browsers.

Sliding Portfolio

The main interaction in the website was sliding through the content in order to being able to explore it with curiosity. So even the Portfolio was build with this approach in mind.
Each slide was like a micro website inside the main one, with fullscreen covers and other interactions within the content.
